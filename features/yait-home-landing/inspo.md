### What makes a crowd delightful, not busy

1. One shape family, varied parameters: every fry is the same rounded-rect silhouette;
   vary only height, hue, and face.
2. Offset timing on a shared motion: everyone bounces, staggered by a few frames with
   different amplitudes — a ripple through the envelope on docking. Synchronized reads
   robotic; fully random reads noisy.
3. Restraint: one focal action at a time (envelope docks, then fries bounce), generous
   negative space, only secondary motion (blinks, tiny leans) elsewhere.

### Vocabulary

Squash and stretch; anticipation; overshoot / follow-through; blob characters;
geometric mascots built from primitives; rounded = likable (shape theory); flat
perspective with pill shadows.

### Hero animation guidance (engineering-blog consensus)

Sources: web.dev animations guide (https://web.dev/articles/animations-guide,
https://web.dev/articles/animations-overview), Josh Comeau
(https://www.joshwcomeau.com/animation/keyframe-animations/,
https://www.joshwcomeau.com/animation/css-vs-javascript/), Codrops SVG clip-path text
reveals (https://tympanus.net/codrops/2024/01/10/experimental-on-scroll-text-animations-with-svg-clip-path/),
Codrops stagger reveals (https://tympanus.net/codrops/2020/06/17/making-stagger-reveal-animations-for-text/),
CSS-Tricks staggered animation (https://css-tricks.com/different-approaches-for-creating-a-staggered-animation/),
CSS-Tricks scroll-driven animations (https://css-tricks.com/unleash-the-power-of-scroll-driven-animations/),
CSS-Tricks SVG vs canvas (https://css-tricks.com/when-to-use-svg-vs-when-to-use-canvas/),
CSS-Tricks reduced motion (https://css-tricks.com/introduction-reduced-motion-media-query/).

- Compositor-only motion: the envelope voyage animates exclusively via
  `transform` (translate/rotate bobbing) and `opacity` — never layout properties.
- Reveal via mask/clip moved by transform: text sits behind a masking layer whose
  transform animates, or the text is an SVG clip-path knockout; per-word staggered
  delays sell the progressive reveal as the envelope passes.
- Time-based intro (plays once on load); if scroll-linked later, native CSS
  `animation-timeline: scroll()/view()` stays off the main thread — no JS scroll
  handlers.
- Crowds via shared keyframes plus per-element `animation-delay` driven by a custom
  property (`--i`, including negative values to start mid-cycle). Dozens of SVG nodes
  are fine; canvas only matters at hundreds-to-thousands.
- Hand-built SVG + CSS transforms beats Lottie (per-frame DOM path mutation) and Rive
  (canvas/WASM runtime) for a scene this size and keeps it fully brand-controllable.
- `prefers-reduced-motion` from day one: under the query, the envelope is already
  docked and the headline fully revealed. A large object traversing the viewport is
  exactly the vestibular-trigger category WCAG flags.
